# EasyShare-Express

一个简单的互传程序.


## 特性

- 化繁为简，一切只要简单的点击几下即可部署.
- 外带依赖性弱，基于Sqlite进行数据存储，不必大费周章部署其他数据库

## 目录
1. [部署](#部署)
2. [先决条件](#先决条件)
3. [开发者自述](#开发者自述)

## 部署
1. 运行`npm install`
2. 启动`npm run start`
3. （默认监听全部地址，和80端口）

### 先决条件
- 有NodeJS环境和NPM或CNPM等包管理器
